# 🎡 Spin Wheel Web App

A modern and interactive **Spin Wheel (Roulette) web application** built with **HTML, CSS, and JavaScript**.
This project features a clean UI, smooth animations, and a fully functional Dark/Light mode system.

---

## 🌐 Live Demo

👉 https://kellernz1.github.io/roulette

---

## 🚀 Features

* 🎯 **Accurate Spin Result**
  The selected option always matches exactly what the pointer lands on.

* 🎨 **Modern UI Design**
  Clean layout with shadows, gradients, and responsive structure.

* 🌙☀️ **Dark / Light Mode**
  Toggle between themes with smooth transitions.
  The selected theme is saved in the browser.

* 📝 **Dynamic Options Input**
  Add, edit, or remove options easily via the textarea.

* 💾 **Local Storage Support**
  Options and theme preferences are saved automatically.

* ❌ **Remove Selected Option**
  After spinning, you can remove the chosen item from the list.

* 🌀 **Smooth Spin Animation**
  Realistic deceleration for a natural spinning effect.
